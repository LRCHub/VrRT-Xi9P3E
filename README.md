# Sunny Girl – 息を切らして

> **歌詞登録ステータス：プレーン歌詞のみ**

```
走る夜の隙間
暮れる光と街
齧るふりをしては
欠ける三日月に笑い眠る
慣れる朝の一人
増える余白と灰
溜める筆を握り
歩むためにまた記す楽譜
走る今日の夜明け
逃げる思い出と陽
僕に背を向けては
爪を切る君に願うばかり
君が朝なら夜更かしもしないのに
仕返しのように君に
言葉を書き集めたのに
読み返して捲ればあまりにも
温度のある文字ばかり
色鉛筆と同じように
大切からなくなるのに
それでも君が描きたい日々を
迷わずに描けるように
さよならは足りないように
いつか君が追いつかないように
```
